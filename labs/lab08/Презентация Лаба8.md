### 2. Нажали клавишу [i] и ввели следующий текст:

    #!/bin/bash
    HELL=Hello
    function hello {
        LOCAL HELLO=World
        echo $HELLO
    }
    echo $HELLO
    hello

### 3. Нажали клавишу [Esc] для перехода в командный режим после завершения ввода текста.
